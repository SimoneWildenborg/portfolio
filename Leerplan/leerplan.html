LEERDOELEN HTML

Ik kan de basisstructuur van een webpagina opzetten (bv parralax, grid, verticaal, horizontaal).
Ik begrijp de opbouw van DOM en kan dit gebruiken in mijn toepassingen.
Ik ken de meeste elementen binnen HTML en kan deze gebruiken.


LEERDOELEN CSS

Ik weet wat absolute / relative is en hoe ik het toe kan passen.
Ik weet hoe de z-index werkt en kan deze toepassen.
Ik kan images bewerken (border, rond maken, grootte, enz).
Ik kan marges / padding toepassen.
Ik kan pagina's modern opmaken.


LEERDOELEN JavaScript

Ik weet wat variabelen, operators, arithmatic-/assignment operaters, data types, statements zijn en kan ze toepassen.
Ik begrijp functions, objects, events, strings, numbers, arrays, boolians, conditions en kan ze toepassen.
Ik begrijp loops: if....else, if....else if, for, while, switch.
Ik weet hoe jQuery werkt en kan het toepassen.

LEERDOELEN Bootstrap

Ik kan een externe link maken naar bootstrap en onderdelen van Bootstrap gebruiken.
Ik kan een hele pagina met Bootstrap opmaken.

LEERDOELEN PHP

Ik weet wat "__construct __destruct" is en kan er mee werken.
Ik weet wat "try..catch..." is en kan het toepassen.
Ik weet wat namespacing is en kan het toepassen.
Ik weet wat een abstract class is en kan het toepassen.

Communication


LEERDOELEN communicatie

Ik vraag hulp als ik ergens niet uitkom.
Ik vraag om extra uitleg als ik iets niet begrijp.

LEERDOELEN Problem solving

Ik probeer eerst zelf de oplossing van mijn probleem te vinden. Lukt dit niet, dan vraag ik hulp.
Ik zoek niet te lang zelf, om zo te voorkomen dat ik achter raak.
Ik probeer zo effectief mogelijk te zoeken naar oplossingen (juiste zoekopdrachten in google, collega studenten vragen).



Time management


LEERDOELEN Time management

Ik kan goed mijn tijd verdelen tussen opleiding en prive, zodat beiden goed blijven lopen.
Als er prive iets belangrijks is, communiceer ik daar over en compenseer de opleidingstijd op een ander moment.
Ik neem voldoende tijd voor mezelf.
